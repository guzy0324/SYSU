输入：一组整数a[1],a[2],...,a[n],n<=2
输出：这组整数中第二大的整数

max_1st_i = 1
i = 2
while i <= n:
    if a[i] > a[max_1st_i]:
        max_1st_i = i
    i += 1
max_2nd_i = 1
if max_1st_i == 1:
    max_2nd_i = 2
i = max_2nd_i + 1
while i <= n:
    if max_2nd_i == max_1st_i:
        continue
    if a[i] > a[max_2nd_i]:
        max_2nd_i = i
    i += 1
return a[max_2nd_i]

输入规模：n
基本操作：两个while内的判断和赋值操作以及while外部的判断、赋值、运算和返回操作。
执行次数最多的操作：第一个while中i += 1
与输入规模的函数关系：n-1
最坏情况下的算法复杂度：O(n)
